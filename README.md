# CosmicLens

**Explore the Universe Through NASA’s Eyes**

CosmicLens is a modern web application that lets users explore NASA’s *Astronomy Picture of the Day (APOD)* with a beautiful space-themed interface and AI-powered explanations.

---

## 🚀 Features

* 📅 **Date Picker** – View APOD images from any date (even your birthday!)
* 🌠 **High-Resolution Images** – Toggle HD quality for stunning visuals
* 🔍 **Fullscreen Mode** – Immersive viewing experience
* 🤖 **AI Explanation** – Get simple explanations of complex space phenomena
* 🎨 **Space-Themed UI** – Dark theme with glowing elements and smooth animations

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** NASA APOD API
* **AI Integration:** Hugging Face / OpenAI API

---

## 📡 API Used

* NASA APOD API: [https://api.nasa.gov/](https://api.nasa.gov/)
  
---

## 🧠 How AI Feature Works

* Fetches the image description from NASA API
* Sends it to an AI model
* Returns a simplified explanation for users
